Set of mini-games for CLI

games in set:

- brain-even
- brain-calc
- brain-gcd
- brain-balance
- brain-progression
- brain-prime

## Install

```
npm install -g jnk-brain-games
```

## Usage

Enter name of the game in console

For example:

```
brain-calc
```
